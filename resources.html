Proximity Sensor

Android Proximity Sensor Example

This page provides an excellent example of how to use the android proximity sensor capabilities. It also links to the general APIs for movement and positioning. The website goes through the process of creating a basic application and displaying either NEAR or AWAY dependent on how close you are to the device, this shows in real time the use of the proximity API. The website also describes the setup needed to use the sensor and how to disable or enable these capabilities dependent on app usage.

Rating: 4/5
